# Image Editor 🎨  

## 📄 Project Description  

**Image Editor** is a modern, web-based image editing application that allows users to upload, modify, and download images directly in the browser. The application provides real-time image adjustments such as brightness, saturation, inversion, grayscale, contrast, hue rotation, sepia effect, and blur intensity. Users can also rotate and flip images with smooth transformations.

The app focuses on clean UI, smooth interactions, and practical JavaScript concepts such as DOM manipulation, CSS filters, image transformations, and Canvas API usage for exporting edited images.

All image processing is performed entirely on the client side without any backend.

---

## ✅ Features  

- Upload image from local device  
- Real-time image preview  

### 🎛 Adjustable Filters (Live Preview)

- Brightness  
- Saturation  
- Inversion  
- Grayscale  
- Contrast  
- Hue Rotate (Color shifting effect)  
- Sepia (Warm vintage tone)  
- Blur (Adjustable blur intensity)

### 🔄 Image Transformations  

- Rotate image:
  - Rotate Left (−90°)  
  - Rotate Right (+90°)  
- Flip image:
  - Flip Horizontal  
  - Flip Vertical  

### ⚙ Additional Functionalities  

- Reset all filters and transformations  
- Download edited image in full original resolution  
- Active filter selection UI  
- Dynamic slider value display  
- Smooth CSS transformations  
- Fully client-side processing (No backend required)  

---

## 🛠 Technologies Used  

- **HTML5** – Page structure and semantic layout  
- **CSS3** – Styling, layout design, and filter effects  
- **JavaScript (Vanilla JS)** – Core logic and DOM manipulation  
- **Canvas API** – Image rendering and download functionality  
- **Font Awesome** – Rotate icons  
- **BoxIcons** – Flip icons
